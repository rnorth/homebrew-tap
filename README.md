# rnorth/homebrew-tap

Personal Homebrew tap.

## Usage

```bash
brew tap rnorth/tap
brew install wyoming-mlx
```

## Formulae

| Formula | Description |
| --- | --- |
| [wyoming-mlx](https://github.com/rnorth/wyoming-mlx) | Apple-Silicon-native TTS and STT for Home Assistant and OpenAI-compatible clients |
